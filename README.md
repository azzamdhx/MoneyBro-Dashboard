# MoneyBro - Dashboard

Aplikasi dashboard untuk manajemen keuangan pribadi yang dibangun dengan Next.js dan teknologi modern lainnya.

## Fitur Utama

- Manajemen transaksi keuangan
- Pelacakan pengeluaran dan pemasukan
- Analisis keuangan visual
- Manajemen hutang dan cicilan
- Antarmuka yang responsif

## Tech Stack

| Technology | Description |
|------------|-------------|
| Next.js 16 | Framework React (App Router) |
| React 19 + TypeScript | UI dan type safety |
| Tailwind CSS | Styling utility-first |
| shadcn/ui | Komponen UI terpilih |
| React Query | Pengelolaan state data/API |
| ESLint | Kualitas dan konsistensi kode |
